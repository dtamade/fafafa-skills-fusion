# Security Policy

## Supported Versions

This project is actively evolving. Please report vulnerabilities against the latest `main` branch unless you have a reason to target an earlier state.

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report privately via:

- Email: `dtamade@gmail.com`

Recommended report content:

1. Vulnerability type and impact.
2. Affected file(s) and version/commit.
3. Reproduction steps or proof of concept.
4. Suggested mitigation (if any).

We will acknowledge receipt as soon as possible and coordinate a fix and disclosure timeline.
