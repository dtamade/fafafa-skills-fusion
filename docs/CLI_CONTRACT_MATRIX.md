## Required machine JSON keys (minimum)

- `release-contract-audit` payload: `schema_version`, `step_rate_basis`, `command_rate_basis`
- `fusion-bridge regression` (contract suite) payload: `schema_version`, `rate_basis`, `total_scenarios`
- cross-platform smoke summary payload: `schema_version`, `platform_label`, `commands_count`, `completed_commands_count`
- remote promotion evidence payload: `schema_version`, `promotion_ready`, `required_jobs`, `missing_jobs`, `failed_jobs`
- Denominator semantics: `step_rate_basis=total_steps`, `command_rate_basis=total_commands`, `rate_basis=total_scenarios`
- Current schema contract: `schema_version=v1`

## Notes

- Contract tests live under `rust/crates/fusion-cli/tests/`.
- If active CLI contracts change, update this matrix together with `rust/crates/fusion-cli/tests/repo_contract.rs` and the relevant shell/CLI contract tests.
- Release gate command bundle is available at `scripts/release-contract-audit.sh`.
- CI gate workflow definition: `.github/workflows/ci-contract-gates.yml`.
- CI machine artifact examples: `/tmp/release-audit-dry-run.json`, `/tmp/runner-suites.json`, `/tmp/runner-contract.json`, `/tmp/cross-platform-smoke-macos/cross-platform-smoke-summary.json`, `/tmp/cross-platform-smoke-windows/cross-platform-smoke-summary.json`.
- Remote promotion evidence artifact example: `/tmp/remote-ci-evidence/remote-ci-evidence-summary.json`.
- Backend failure artifact: `.fusion/backend_failure_report.json`.
- Backend failure JSON should preserve the routing context fields `backend_status`, `backend_primary`, and `backend_fallback` for machine inspection.
- `.fusion/task_plan.md` reviewer gate metadata is `Review-Status: none|pending|approved|changes_requested`.
- `fusion-status.sh --json` should preserve stable machine fields for automation, including `workflow_id`, `goal`, `understand_mode`, `understand_forced`, `understand_decision`, `codex_session`, `claude_session`, `planner_codex_session`, `planner_claude_session`, `coder_codex_session`, `coder_claude_session`, `reviewer_codex_session`, `reviewer_claude_session`, `runtime_state`, `guardian_status`, `guardian_no_progress_rounds`, `guardian_same_error_count`, `guardian_wall_time_ms`, `safe_backlog_last_added`, `safe_backlog_last_injected_at_iso`, `runtime_last_event_id`, `runtime_scheduler_enabled`, `agents_enabled`, `agent_mode`, `agent_explain_level`, `agent_current_batch_id`, `agent_active_roles`, `agent_current_batch_tasks`, `agent_review_queue`, `agent_review_queue_size`, `agent_last_decision_reason`, `agent_batch_reason`, `agent_collaboration_mode`, `agent_turn_role`, `agent_turn_task_id`, `agent_turn_kind`, `agent_pending_reviews`, `agent_blocked_handoff_reason`, `agent_selected_reasons`, `agent_blocked_reasons`, and `agent_review_reasons` when the corresponding runtime/session data exists.
